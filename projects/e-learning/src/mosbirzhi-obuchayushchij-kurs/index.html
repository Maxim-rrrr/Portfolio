Описание

Создали онлайн-курс для начинающих инвесторов. Задачей нашей команды была разработка концепции, дизайна и иллюстраций, всего текстового контента, анимации и 2D-роликов.

Курс реализован в формате сайта и состоит из четырех модулей:
  1. Принципы финансового планирования и инвестирования.
  2. Разбор и аналитика инвестиционных продуктов.
  3. Рекомендации для совершения сделок без ошибок.
  4. Советы по защите прав инвестора.

Сайт проекта находится в открытом доступе. Любой желающий может бесплатно узнать о базовых инструментах инвестирования, составить план для достижения своих финансовых целей и начать пользоваться услугами биржевой площадки.
